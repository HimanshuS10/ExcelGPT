# ExcelGPT:

Turn spreadsheets into insights with AI.
Website Link: https://excelgpt.vercel.app/

# Tech Stack:

**Frontend:**
- React.js
- TailwindCSS

**Backend:**
- Python Flask
- Gemini API
- Pandas

### Deployment
- Vercel (Frontend)
- Render (Backend API)

# Running the program:

`git clone <URL>`

# Running the frontend:
`cd frontend`
`npm i`
`npm run dev`

# Running the backend:
`python -m venv venv` (Windows) `python3 -m venv venv` (Mac) 
`venv\Scripts\activate` (Windows)  `source venv/bin/activate` (Mac)
`pip install -r requirements.txt`
`python app.py`
